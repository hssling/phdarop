Map of funders to projects, with each funder's fit, instrument, and the flagship/project it
should back. The Centre is financed as a **portfolio**, not a single grant: one anchor
institutional grant + per-flagship project grants + a deployment contract.

## 3. Funding architecture (compressed)

```
Wave 0 (sprint):   Anchor institutional grant (ICMR/DHR or Wellcome)  ─► stand up cores
                 + 1 flagship project grant (Gates TB or Bank workforce)
Wave 1 (~1y):    + 2 flagship grants + platform grant (UKRI/Horizon)
                 + World Bank advisory/deployment contract (revenue + adoption)
Wave 2 (2-3y):   + capstone (P38) + multi-country consortium (Horizon) + NIH training (D43)
```

## 4. Why funders will say yes (the pitch per funder type)

- **Mission funders (Gates/Global Fund):** "Same budget, ~40% more DALYs averted" (computed AII
  = 0.29, [E09](engines/E09-optimization-engine.md)) — pure allocative efficiency, no new money.
- **System funders (World Bank/DHR):** a reusable, sovereign, open decision-support platform that
  outlives any single project and strengthens government capacity.
- **Science funders (Wellcome/UKRI/NIH/Horizon):** a *new field* with novel methods (CFA,
  optimizer⇄twin loop) and a 100-manuscript pipeline — durable scientific return.
- **Normative bodies (WHO):** standards, global public goods, and One-Health decision
  infrastructure.

## 5. Sustainability

Transition from grants to a **blended model**: government deployment contracts (decision-support
as a service), philanthropic core support, and competitive science grants — so the Centre is
not single-funder-dependent. Open-source platform lowers adoption cost and widens the funder base.
